# v1.0.2 - 2025-07-06

## 개선 사항 (Improvements)
*   `PasswordWindow` 실행 시 비밀번호 입력 필드에 자동으로 포커스가 가도록 수정.
*   경기번호 계산기 "강수" 열 정렬 로직 개선:
    *   숫자 값은 선택된 정렬 방향(오름차순/내림차순)에 따라 올바르게 정렬되도록 수정.
    *   "선"이 포함된 문자열은 일반 문자열 정렬을 따르며, 숫자 값보다 뒤에 정렬되도록 유지.
